### Python Basics

#### Primitive expressions

原始表达式，只需要一步就可以完成赋值

```
>>>3
3
```



#### Arithmetic expressions

```python
# 乘方运算符	exponentiation **
>>> 2 ** 5
32

# 浮点除法 float point division	/
>>> 9 / 7
1.2857142857142858

# 取整运算 Floor division //
>>> 9 // 7
1

# 取模运算 Modulo %
>>> 
```

#### Assignment statements

python 没有赋值的变量不会被记录

```python
x = 20
x + 2
>>> x
return 20
```

**Names are bound to values, not expressions** 

只能通过 `=` 来绑定 `name` 与 `value`
